# Caderno de Estilos

Caderno de estilos de Design Hipermédia

## Componentes

1. **Alternar entre tema claro/escuro** - Usando `light-dark()`
2. **Cores** - Usando variáveis CSS com `var()`
3. **Tipografia** - Estilização de `h1-h6` e `p`
4. **Accordion** - Usando `details` e `summary`
5. **Alerts** - Usando SVG para ícones
6. **Badge** - Usando Anchor positioning
7. **Breadcrumb** - Usando `ol` e `li`
8. **Buttons** - Usando elemento `button`
9. **Card** - Usando container query
10. **DropDown** - Usando popover e anchor positioning
11. **Formulários** - Usando `form`, `label`, `input`
12. **Listas** - Usando `ul` e `li`
13. **Modal** - Usando popover, dialog e backdrop
14. **Navbar** - Usando elemento `nav`
15. **Navs & tabs** - Usando `details`, `summary` com `attr()`, `grid`, `subgrid`, `:is()`, `::details-content`
16. **Pagination** - Usando `nav`, `ul`, `li`
17. **Progress** - Usando elemento `progress`
18. **Spinners** - Usando animações CSS
19. **Tabelas** - Usando elemento `table`
20. **Tooltip** - Usando popover e anchor positioning
